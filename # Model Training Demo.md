# Model Training Demo

This repository contains a Jupyter Notebook demonstrating a simple linear regression model using synthetic data. The notebook includes:
- Data generation and visualization
- Saving the dataset to a CSV file
- Preparing the data for linear regression

## Files
- `main.ipynb`: The Jupyter Notebook with the code for data generation and linear regression.
- `linear_regression_data.csv`: The generated dataset saved as a CSV file.

## Requirements
- Python 3.7+
- Libraries: `numpy`, `matplotlib`, `pandas`, `scikit-learn`

## Usage
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebook:
   ```bash
   jupyter notebook main.ipynb
   ```
